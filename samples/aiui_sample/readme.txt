1. compile:in aiui directory,for linux 64:source 64bit_make.shfor linux 32:source 32bit_make.sh 2. runcd build./demoNote:1. The library is compiled in Ubuntu 16.10 x64 and Ubuntu 16.04 LTS i686.Please compile and test the demo in the same environment.2. If error 10116 is reported, please make sure the 'scene' in build/AIUI/cfg/aiui.cfg issame with the scene name of your application on aiui.xfyun.cn.3. The file encoding format must be utf-8.4. If you want to use ivw for wakeup,  please set the wakeup_mode to "ivw" and res_path to wakeup source file in aiui.cfg
